Integrated Open Data Sources

GramSetu leverages verified, official Indian and global database repositories to automate its metrics. Below are details and references to the integrated data platforms.

OGD Platform India (Data.gov.in)

Used to fetch real-time and daily-updated agricultural market arrival rates and prices for essential grains in various states.

Refresh Rate: Weekly (automated Scheduler)

Visit Data.gov.in API

Dept of Animal Husbandry & Dairying (DAHD)

National livestock health database of epidemic outbreaks, cattle disease reports, and mandatory vaccination guidelines (under the National Animal Disease Control Programme).

Refresh Rate: Monthly (automated Scheduler)

Visit DAHD India Portal

IUCN Red List of Threatened Species

Used to determine the global threat level and extinction status of avian wildlife species identified via microphone recordings.

Refresh Rate: Permanent static catalog reference

Visit IUCN Red List

OpenStreetMap and Leaflet

Provides the cartography engine that maps livestock infection points, vaccination clinics, and citizen-contributed wild bird sound detection pins.

Refresh Rate: Dynamic real-time client-side rendering

Visit OpenStreetMap
